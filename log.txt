git clone: comando utilizado no terminal para copiar um repositório remoto para uma máquina local.
git add: comando que prepara as suas alterações (ficheiros criados, modificados ou apagados) para serem salvas no Git, movendo-as para uma "área de espera" (staging area). Ele avisa ao Git: "quero incluir estas mudanças no próximo registo" (commit), permitindo selecionar o que guardar
git commit: guarda as alterações no histórico do repositório.
